# Cifra Atbash

Implementação da **cifra Atbash** em Python, com um menu no terminal para **criptografar** e **descriptografar** mensagens usando o alfabeto (a–z).

A Atbash é uma cifra alfabetica onde os caracteres são espelhados:

- `a ↔ z`
- `b ↔ y`
- `c ↔ x`
- ...

Como a substituição é simétrica, **o mesmo mapeamento serve para criptografar e descriptografar** (neste projeto, mantive duas funções separadas por didática).

---

## Requisitos

- Python 3.x

---

## Como executar

1. Salve o código em um arquivo, por exemplo:
   - `atbash.py`

2. Execute no terminal:

```bash
python atbash.py
